Module 1. JavaScript Fundamental

This module introduce the fundamental concepts of JavaScript syntax. It covers from the basic JS programming concept, such as data type and variables, to more advanced topics as objects and methods in JS. The content material of module 1 is structure as:

🕗8hr to complete module

📑6 lectures

Module Objectives

  • Students will focus on JavaScript programming development.
  • Students integrate knowledge and skills to develop HTML, CSS, and JavaScript into their lab assignments and project.

Learning Outcomes

Upon completion of this module materials, students will be able to:

  • Demonstrate proficiency in utilizing JavaScript to collect user input through the prompt method.
  • Apply various string and array methods to manipulate and analyze the collected word and/or data.
  • Develop problem-solving skills through the implementation of JavaScript code to achieve specified tasks, fostering a deeper understanding of fundamental programming concepts.
  • Analyze problem scenarios and select appropriate conditional statements to effectively manage program flow and behavior.
  • Develop and evaluate control flow programs through iterative testing and debugging processes, fostering the enhancement of critical thinking and problem-solving skills in JavaScript programming.
  • Define and implement JavaScript functions to perform specific tasks, including error handling and mathematical operations.
  • Evaluate and refine the implemented functions through iterative testing and troubleshooting, ensuring functionality and addressing any potential errors or edge cases.
  • Construct JavaScript objects using constructor functions, enabling the creation of reusable templates for defining similar types of objects.
  • Apply the "this" keyword within constructor functions and methods to reference the current object instance, enabling access to object properties and methods.
  • Evaluate and refactor object-oriented JavaScript code through iterative testing and debugging processes, fostering a deeper understanding of object-oriented programming concepts and best practices.
  • Utilize different types of loops, including "for," "while," and "do-while," to perform repetitive tasks and manipulate data structures based on specific conditions.
  • Employ loop iteration techniques such as incrementing, decrementing, and iterating over array elements or object properties to access and manipulate data effectively.
  • Evaluate problem scenarios and select appropriate loop constructs and iteration strategies to efficiently solve programming challenges, fostering critical thinking and algorithmic problem-solving skills.
